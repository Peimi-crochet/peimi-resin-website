# PEIMI Resin Website V2

A static GitHub Pages-ready website inspired by modern Shopify-style resin serveware stores, rebuilt with original PEIMI branding, original copy, and CSS-generated placeholder product visuals.

## Deploy on GitHub Pages
1. Upload all files to a GitHub repository.
2. Go to Settings > Pages.
3. Choose branch `main` and root folder.
4. Save and open the generated Pages URL.

## Important
- Replace `your-email@example.com` in `contact.html`.
- The forms are static. Connect Formspree, Netlify Forms, Shopify, WooCommerce, or your own backend to receive messages.
- Replace CSS placeholder product art with your real product photos before launching paid ads.
